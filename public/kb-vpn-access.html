VPN Access

Connectivity, certificates, split-tunnel rules.

VPN connectivity checklist

  • Confirm internet access (ping/public site)
  • Verify correct gateway/region
  • Disable conflicting proxy/ad-block/VPN

Reinstall profile / renew cert

  • Remove old VPN profile
  • Download latest client/profile
  • Renew client certificate; restart service

Split-tunnel rules

  • Check route table for corporate subnets
  • Add exclusions for heavy media sites
  • Validate DNS resolvers while on VPN

When to escalate

  • Repeated disconnects across sites
  • Certificate enrollment failures
  • Latency/packet loss on tunnel path
